Step 7 : Format, Step, Reset Memory, Reset Registers, Toggle Breakpoint:
Use the FORMAT button if you need to clear or prepare the input area or memory for a new program.


Step Button (stepButton): This button advances the VM to the next instruction and executes it. The vm.step() method handles this


Reset Memory Button (resetMemoryButton): This button resets the memory to its default state (in this case, zeros). The vm.resetMemory() method takes care of this.


Reset Registers Button (resetRegistersButton): This button clears all registers. The vm.resetRegisters() method is responsible for this.


Toggle Breakpoint Button (breakpointButton): This button toggles a breakpoint at the current instruction index, allowing you to pause execution at specific points. The vm.toggleBreakpoint() method is used for this.
